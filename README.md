Portfolio Management System

A dynamic, database-driven portfolio website with a comprehensive admin panel for content management. Built with PHP, MySQL, HTML/CSS, and JavaScript.
Frontend
Responsive Design - Works on all devices (desktop, tablet, mobile)
  Dynamic Content- All content managed through admin panel
  Project Showcase- Display projects with images, descriptions, and technologies
Skills Section- Visual representation of technical skills with progress bars
  Contact Form- Visitors can send messages directly through the website
Admin PanelSecure Authentication- Login system with password hashing
  Dashboard*- Overview with statistics (projects, messages, etc.)
  About Management Update bio and profile image
  Skills Management - Add, edit, delete skills with proficiency levels
  Project Management- Full CRUD operations for projects
  Message Management - View and manage contact form submissions
  Technology Stack
Backend:PHP
  Database:MySQL
  Frontend: HTML, CSS, JavaScript
